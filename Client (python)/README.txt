Клиент, входящий в состав коплекса шпионского ПО. Работает на всех версиях ОС, но в целом комплекс работает
на ОС Windows 7.
Клиент выполняет следующие функции:
1 Подключение к центру управления на базе FTP сервера, используя TLS;
2 Скачивание модулей шпиона и необходимых файлов;
3 Осуществление управления модулем шпиона;
4 Загрузка всех собранных данных шпионом на сервер FTP.

Сборка скрипта на языке Python в exe-файл осуществляется при помощи py2exe (setup.py приложен).
