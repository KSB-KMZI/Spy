Модуль входящий в состав комплекса шпионского ПО v1.0.0. Работает на ОС Windows 7 и выше
Модуль представляет собой программу для снятия состояния экрана

Для запуска модуля безрежима централизованного управления достаточно просто запустить 
бинарный файл, при этом если не был прикреплён файл конфигурации Sconfig.ksb, то будет принята последняя
успешная конфигурация. Модуль в состоянии работать автономно, то есть и при отсутствии
файла конфигурации как такового. Модуль в режиме реального времени отслеживает изменения
конфигурации в файле конфигурации, а также наличие этого самого файла конфигурации. Все 
изменения конфигурации вступают в силу только после того, как модуль отработает предыдущую
конфигурацию, кроме первого параметра, который завершает процесс самого модуля.

Для создания файла конфигурации используйте программу ConfigEditor.exe

Для создания или изменения файла конфигурации в ручном режиме требуется знать что означает каждый 
из параметров файла Sconfig.ksb

Первый параметр (0 или 1) 0 - процесс работает, 1 - процесс завершён
Восьмой параметр (0 или 1) 0 - модуль не запускается, 1 - модуль запускается
Третий параметр (0 - 32767) - Интервал снятия состояния экрана

При централизованном управлении смотрите описание для клиента на стороне жертвы
